## About

Cyber​​ Security University 是一个精选的免费教育资源列表，专注于边做边学.

这有 6 个部分. 简介和预安全、免费初学者红队路径、免费初学者蓝队路径、奖金实践/最新的 CVE 和极其困难的房间. 这些任务在难度上是线性的. 所以建议按顺序进行. 但是如果您发现您已经熟悉这些概念，您仍然可以跳来跳去并跳过一些房间.
