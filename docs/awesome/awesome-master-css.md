&gt; 根据类名自动生成对应的CSS规则，增强的CSS语法，可以用更少的代码构建UI，直接在生产中使用高性能的JIT. 

[Master CSS](https://css.master.co) 是一种具有增强语法的虚拟 CSS 语言，仅使用 HTML 即可高效地构建您的 UI 和设计系统.

## Plugins 
- [@master/normal.css](https://github.com/master-co/css/tree/beta/packages/normal.css) - 标准化浏览器的样式.
- [@master/keyframes.css](https://github.com/master-co/css/tree/beta/packages/keyframes.css) - 简单实用的 CSS 关键帧.
- [@master/style-element.react](https://github.com/master-co/style-element.react) - 使用条件类名称快速创建样式化的 React 元素.
- [@master/literal](https://github.com/master-co/literal) - 有条件地使用模板文字构造类名和字符串.
